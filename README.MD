# LINE 通報格式產生器 (LINE Report Generator)

**一鍵產出格式整齊、視覺醒目的 LINE 工作群組通報訊息。**

這個專案是一個輕量級的靜態網頁工具，旨在解決在 LINE 溝通時資訊雜亂、格式不一的問題。透過網頁表單填寫，即可自動產出帶有區域區隔感與重點標註的 LINE 專屬格式。

## 核心功能

- **模組化架構**：採用 SOLID 原則重構，將資料格式化邏輯與 UI 控制分離，確保程式碼易於維護與擴充。
- **重要性標記**：提供一鍵勾選「標記為重要」，在通報最上方自動加入顯眼的提醒文字。
- **智慧期限輔助**：日期選擇器搭配雙下拉選單（日期補充、時間補充），可快速產出如「(明天) 下班前」等具體期限說明。
- **彈性繳交方式**：支援設定「(都要)」或「(任選一種)」，明確溝通任務要求。
- **條列式動態備註**：支援動態增加多列自定義備註，輸出時自動轉換為 1. 2. 3. 序號排列之條列式格式。
- **自動複製功能**：點擊產生按鈕後自動將結果寫入剪貼簿，並提供按鈕狀態回饋。
- **深色模式支援**：自動感應系統主題，提供符合閱讀舒適度的深色介面。

## 使用技術

- **HTML5**: 結構化標記。
- **CSS3**: 使用 CSS 變數、Flexbox 與 Grid 佈局，並包含 Media Queries 處理深色模式。
- **JavaScript**: 原生 Vanilla JS 實作，不依賴外部框架。

## 使用步驟

1. 填寫「通報單位」與必填的「案由」。
2. 設定「繳交期限」，並視需求選擇日期與時間的補充說明。
3. 選擇「繳交方式」及其完成要求（都要或任選）。
4. 勾選預設備註，或點擊按鈕增加多行自定義備註內容。
5. 點擊 [產生並自動複製] 按鈕。
6. 在 LINE 對話框貼上並傳送。
