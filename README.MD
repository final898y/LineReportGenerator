# LINE 通報格式產生器 (LINE Report Generator)

這是一個輕量級的靜態網頁工具，旨在解決 LINE 群組溝通時資訊雜亂、格式不一的問題。透過網頁表單填寫，即可自動產出帶有區域區隔感與重點標註的專業通報格式。

## 核心功能

- **模組化架構**：遵循 SOLID 原則進行重構，將資料收集、格式化邏輯與 UI 控制完全解耦，提升系統的可維護性與擴充性。
- **多模式切換**：支援一般案件通報、週報會議、公告通知等不同場景的格式產生。
- **重要性標記**：提供一鍵勾選功能，自動在通報頂端加入醒目的「重要」標示。
- **智慧期限輔助**：日期選擇器整合預設標籤（如：今天、下班前、本週內），產出具備口語化描述的期限資訊。
- **動態備註系統**：支援動態增減備註項目，輸出時自動進行編號並條列化。
- **自動複製機制**：產生結果後自動寫入剪貼簿，並提供即時的視覺操作回饋。
- **深色模式支援**：支援系統層級的主題切換，優化弱光環境下的使用體驗。

## 使用技術

- **HTML5**: 結構化頁面標記。
- **CSS3**: 採用 CSS 變數、Flexbox 與 Grid 佈局，並透過 Media Queries 實作回應式設計與深色模式。
- **JavaScript (ES6+)**: 採用原生 Vanilla JS，不依賴任何第三方框架，並使用 ES Modules 進行模組化開發。

## 使用步驟

1. 選擇適用的通報模式（一般、週報、公告）。
2. 填寫案由、期限、詳細說明等必要資訊。
3. 根據需求勾選預設備註，或點擊按鈕增加自定義備註。
4. 點擊 [產生並自動複製] 按鈕，系統將自動產生格式並完成剪貼簿複製。
5. 在 LINE 對話框貼上並傳送。

## 開發者備註

本專案採用策略模式（Strategy Pattern）處理不同的報告模式。若需擴充新模式，僅需在 `js/report-modes.js` 中定義對應的策略類別，並於 `index.html` 增加 UI 控制項，不需修改核心協調邏輯。
