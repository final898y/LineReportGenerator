# LINE 通報格式產生器 (LINE Report Generator)

🚀 **一鍵產出格式整齊、視覺醒目的 LINE 工作群組通報訊息。**

這個專案是一個輕量級的靜態網頁工具，旨在解決在 LINE 手機版與電腦版溝通時，資訊雜亂、格式不一的問題。透過網頁表單填寫，即可自動清洗資料並產出帶有 Markdown 風格的 LINE 專屬格式。

## ✨ 特色功能

- **自動資料清洗**：自動移除輸入內容前後的空白與多餘換行（Sanitization）。
- **條件式渲染**：類別選「無」時自動隱藏該行，保持訊息精簡。
- **視覺強化**：利用 LINE 電腦版支援的特殊語法（紅字、區隔線）強化資訊階層。
- **響應式設計 (RWD)**：支援手機、平板與電腦操作。
- **智慧時間選取**：內建日期時間選取器，自動轉換為易讀格式。
- **一鍵複製**：產生後自動複製到剪貼簿，操作流暢。

## 🛠 使用技術

- **HTML5**: 結構化標記與原生日期元件。
- **CSS3 (Flexbox/Grid)**: 響應式佈局與自定義變數系統。
- **JavaScript (Vanilla JS)**: 字串樣板處理、DOM 操作與剪貼簿 API。

## 🚀 如何使用

1. 開啟網頁介面。
2. 填寫「案由」、「通報單位」與「繳交期限」等資訊。
3. 勾選所需的「繳交方式」與「相關備註」。
4. 點擊 **[產生並自動複製]** 按鈕。
5. 到 LINE 群組按下 **貼上 (Ctrl+V)** 即可發送。
